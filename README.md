🎯 Valorant Agent Guesser

A small web app where users try to guess Valorant agents based on hints. Built using JavaScript and a public API to make it interactive and fun.

📌 What this project does

    Shows hints about a Valorant agent (like role, abilities, or image)
    User has to guess the correct agent
    Checks if the guess is correct or not
    Also allows exploring all agents and their details

🚀 Features

    Guess the agent using given hints
    Fetches agent data from API
    Search agents by name
    Filter agents by role (Duelist, Sentinel, etc.)
    Sort agents alphabetically
    Click to view more details
    Responsive design

🧠 How the guessing works

    An agent is selected randomly
    Some hints are shown to the user
    User selects or types a guess
    The app checks the answer and shows result

⚙️ Current scope

    Implemented:
      API integration using fetch
      Random agent selection
      Basic guessing logic
      Search, filter, and sort
      UI to display agents
